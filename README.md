# Hardhat Sample Project

## Development

### VSCode - Remote Container（Recommended）

**Pre-requisites**

- Install Docker Desktop
- Install VSCode

**Quick start**

- Command Palette（Mac: ⌘ + Shift + P, Win: ctrl + Shift + P）
- Type - Open Folder in Container
- Run `yarn install`

### Local

**Pre-requisites**

- Install Node.js v16 latest
- Install yarn via npm

**Quick start**

- Run `yarn install`
